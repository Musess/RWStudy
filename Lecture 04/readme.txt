Class Diagram

플레이어 
(체력, 크기, 위치) 
[플레이어의 위치는 고정, 플레이어의 크기는 반지름 20, 플레이어의 체력은 3]

적
(체력, 위치, 오는방향, 크기, 데미지)
[적의 체력은 항상 랜덤 (10부터 시작), 적의 위치는 항상랜덤, 적의 오는 방향은 플레이어의 위치, 적의 크기는 체력비례, 적이 플레이어에게 주는 데미지는 1]

총알
(총알 수, 데미지)
[총알을 발사 할 때는 마우스 클릭을 입력 받아 발사 , 총알 수는 3발 , 총알 데미지는 10, 총알이 화면에 사라질때까지 3발 이상 존재할 수 없음]

게임 종료 조건
1. 플레이어의 체력이 0으로 떨어졌을 때
2. 점수가 100점 이상일때

점수 흭득 조건

적을 총알로 처치할 시 1점 흭득


Flowchart

게임 루프 시작 -> 점수 100점 이상 YES-> 게임 루프 종료

게임 루프 시작 -> 플레이어 체력 0인가? YES -> 게임 루프 종료

게임 루프 시작 -> 점수 100점 이상 NO-> 게임 루프 반복

게임 루프 시작 -> 플레이어 체력 0인가? NO -> 게임 루프 반복
